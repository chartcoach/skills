# chartcoach skills

[![skills.sh](https://skills.sh/b/chartcoach/skills)](https://skills.sh/chartcoach/skills)

chartcoach skills help agents critique charts, recommend encodings, compare
design options, and explain chart feedback with sources.

```bash
npx skills add chartcoach/skills
codex 'Hey $chartcoach, how should I show uncertainty for a general audience?'
```
